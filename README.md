# Sharp bakery

## Contributers
* Robert Bryan

## Description
Basic price calculator using c#

## Technologies Used
* C#
* .Net6
* MsTest
* Spectre

## Setup/Installation Requirements

### Requirements
* dotnet >= 6.0.402

### Setup
inside BakeryApp run:
```
dotnet restore
```

### Run
inside BakeryApp run:
```
dotnet run
```

### Build
inside BakeryApp run:
```
dotnet build
```
